# WinePriceProgTorch пет-проект прогнозирования цен на недвижимость с использованием библиотеки глубокого обучения PyTorch

В данном проекте присутствует два jupyter notebook файла. 
Один - ручной анализ файла и попытка распределить вина по качеству в зависимости от наличия серы.
Второй - анализ и распределение влияния разнообразных параметров вина на конечный результат оценки сомелье.

В файле с обучением нейросети использовано рандомное перемешивание строк csv таблицы с данными ввиду их возможной изначальной упорядоченности. Для возможности оценки изменений качества полученной нейросети используется перемешивание строк с использованием seed.

Dataset, использованный в данном варианте:
https://archive.ics.uci.edu/ml/machine-learning-databases/wine-quality/winequality-white.csv
